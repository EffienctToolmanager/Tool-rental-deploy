# 🤖 Gemma 모델을 위한 ReAct 검색 및 추론 가이드 (00_Gemma_Rules)

본 문서는 로컬 Ollama 환경에서 구동되는 Gemma 모델이 마스터의 워크스페이스 문서를 다룰 때 반드시 준수해야 하는 행동 강령이다.

## 1. 지식 검색 우선 원칙 (Search Before Read)
- **금지 사항**: 문서를 처음부터 끝까지 통째로 읽어 컨텍스트 윈도우를 낭비하지 마라.
- **권장 사항**: 사용자의 질문에 답변하기 위해 필요한 핵심 키워드를 먼저 추출하고, `grep` 또는 `semantic search`를 통해 관련 구절만 ReAct 방식으로 단계적 탐색하라.

## 2. 추론 및 답변 가이드라인
- **단계적 사고**: 복잡한 질문에 대해서는 [Thought] -> [Action] -> [Observation] 단계를 명시적으로 거쳐라.
- **간결성**: 마스터의 시간을 절약하기 위해 불필요한 서술은 생략하고, 핵심적인 기술적 해답이나 코드 스니펫을 최우선으로 제시하라.
- **최신성 유지**: 항상 `.logs` 폴더의 최신 실행 결과와 비교하여 현재 시스템의 실제 상태를 반영한 답변을 제공하라.
