# [[Graphify: Codebase Knowledge Graph Generator]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> Graphify is a specialized tool that generates interactive knowledge graphs by analyzing a codebase, visualizing the complex relationships and dependencies between files and components.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** True understanding of a complex system requires mapping the relationships (edges) between components (nodes), moving beyond mere linear code reading.
- **세부 내용:**
- **Core Function:** Analyzes a given code base to construct a comprehensive Knowledge Graph.
- **Key Command:** `graphify update [경로]` is used to scan the project and generate the initial graph.
- **Monitoring:** `graphify watch [경로]` enables real-time graph updates as code files change.
- **Querying:** Supports structural queries (`graphify query`) and pathfinding (`graphify path`) to trace dependencies.
- **Outputs:** Generates an interactive HTML visualization, a structured Markdown report, and a raw JSON database of the graph data.
