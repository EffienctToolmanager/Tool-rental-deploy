# 사내 공구 대여 시스템(Tool Rental Web) 구축 가이드

이 문서는 React 기반의 사내 비품/공구 대여 앱을 Microsoft Teams에 통합하기 위해, 사용자가 Microsoft 365 환경에서 수동으로 구성해야 할 인프라(DB, 인증, 매니페스트) 설정 방법을 안내합니다.

---

## 1. 데이터베이스 구성: SharePoint List 세팅
앱이 데이터를 저장할 수 있도록 회사 SharePoint 사이트에 접속하여 **새 목록(List)**을 생성해야 합니다.

- **목록 이름**: `ToolRentalDB` (영문 권장)
- **필수 컬럼 구성**:
  1. `Title` (기본 생성됨) - **용도**: 장비명 (단일 줄 텍스트)
  2. `Status` - **용도**: 대여 상태 (선택 항목: `대여가능`, `대여중`, `수리중`)
  3. `BorrowerName` - **용도**: 대여자 이름 (단일 줄 텍스트 또는 사용자 필드)
  4. `BorrowerEmail` - **용도**: 대여자 이메일 (단일 줄 텍스트)
  5. `CheckoutDate` - **용도**: 대여 일시 (날짜 및 시간)
  6. `ReturnDate` - **용도**: 반납 일시 (날짜 및 시간)
  7. `ImageURL` - **용도**: 장비 사진 링크 (하이퍼링크 또는 단일 줄 텍스트)
  8. `Spec` - **용도**: 규격 및 상세 설명 (여러 줄 텍스트)

*(작업 완료 후 해당 SharePoint 사이트의 URL을 `.env` 파일에 넣게 됩니다.)*

---

## 2. 보안 인증: Microsoft Entra ID (구 Azure AD) 앱 등록
Teams 내에서 사용자가 자동으로 로그인(SSO)되도록 앱을 등록합니다.

1. [Microsoft Entra 관리 센터](https://entra.microsoft.com/)에 접속합니다.
2. 좌측 메뉴에서 **[앱 등록 (App registrations)]** -> **[새 등록]**을 클릭합니다.
3. **이름**: `Tool Rental Teams App` (자유롭게 입력)
4. **지원되는 계정 유형**: 이 조직 디렉터리의 계정만 (단일 테넌트)
5. **리디렉션 URI (선택 사항)**: 
   - 플랫폼: `단일 페이지 애플리케이션(SPA)`
   - URL: 로컬 테스트를 위해 `http://localhost:5173` 입력 (추후 실제 배포 주소로 변경)
6. 등록 버튼을 누릅니다.
7. 생성된 화면에서 **[애플리케이션(클라이언트) ID]** 와 **[디렉터리(테넌트) ID]** 값을 복사해 둡니다.

**API 권한 부여:**
- 왼쪽 메뉴 **[API 권한]** 클릭 -> **[권한 추가]** -> **[Microsoft Graph]** -> **[위임된 권한]**
- `User.Read` (기본), `Sites.ReadWrite.All` (SharePoint 접근용) 권한을 체크하고 추가합니다.
- (필요시) `[조직 이름]에 대한 관리자 동의 부여` 버튼을 눌러 승인합니다.

---

## 3. Teams Tab 매니페스트(Manifest) 패키징
앱을 Teams 좌측 탭이나 채널 탭에 삽입하기 위해 매니페스트 파일이 필요합니다. 
이 과정은 프론트엔드 개발이 끝난 후, 코드로 만들어진 `manifest.json` 파일과 아이콘 이미지 2개(192x192, 32x32)를 `.zip`으로 압축하여 Teams의 **[앱 관리] -> [사용자 지정 앱 업로드]** 메뉴를 통해 진행합니다.

---

## 4. 디자인 테마 (브랜드 가이드라인 이식)
디자인은 CSS 변수(Variables)를 통해 쉽게 제어할 수 있도록 구조화됩니다. 
향후 코드 내의 `App.css` 최상단 `:root` 영역에서 회사의 브랜드 로고 설명서에 맞는 색상 코드로 수정만 하면 전체 테마가 자동으로 변경됩니다.

```css
:root {
  --primary-color: #0047AB; /* 회사 메인 브랜드 컬러로 변경하세요 */
  --secondary-color: #12192a;
  --accent-color: #64ffda;
  --logo-padding: 20px;
  --border-radius: 4px;
}
```
