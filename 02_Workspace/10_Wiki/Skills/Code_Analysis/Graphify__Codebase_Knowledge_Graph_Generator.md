# [[Graphify: Codebase Knowledge Graph Generator]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> Graphify is a specialized tool that analyzes code repositories to build and visualize a comprehensive knowledge graph of file dependencies and relationships.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** Understanding system architecture requires converting physical code structure into an explicit, queryable, and visualized graph model.
- **세부 내용:**
- **Purpose:** Generates a Knowledge Graph from a codebase to map complex relationships between files and components, registering as a core analysis skill.
- **CLI Commands:** Key commands include `update` (scans and generates the graph), `watch` (real-time monitoring of changes), `query` (structural questions), and `path` (finding the shortest call path between two nodes).
- **Outputs:** Produces three main assets: an interactive HTML graph (`graph.html`), a summary report (`GRAPH_REPORT.md`), and the raw JSON database (`graph.json`).
- **Integration:** Supports Obsidian connectivity via the `tree` command for parallel visualization of physical folder structure and logical relationships.
