# [[Graphify: Code Knowledge Graph Generation and Analysis Guide]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> Graphify is a core anti-gravity skill tool that analyzes a codebase to generate, visualize, and query a comprehensive knowledge graph of file dependencies and relationships.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** Codebase structure and dependencies can be effectively modeled and queried using graph database principles, aiding complex system understanding.
- **세부 내용:**
- Graphify generates a Knowledge Graph (KG) by analyzing the codebase, making complex file relationships visible.
- Key CLI commands include `graphify update [path]` for initial scanning and `graphify query "question"` for structured queries.
- It supports real-time monitoring (`graphify watch`) and finding shortest dependency paths (`graphify path`).
- Output includes an interactive HTML visualization (`graph.html`), a summary report (`GRAPH_REPORT.md`), and raw JSON data (`graph.json`).
- The tool can be integrated with Obsidian, using tree views to map physical folder structure alongside conceptual knowledge graph data.
