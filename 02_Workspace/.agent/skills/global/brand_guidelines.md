# 🎨 회사 브랜드 내규 (Brand Guidelines)

이 스킬은 UI, 프론트엔드, CSS 코드를 작성할 때 **가장 우선적으로** 적용되어야 합니다.

## 1. 핵심 컬러 팔레트 (Core Colors)
- **Primary Color**: `#0047AB` (Company Blue)
- **Secondary Color**: `#F5A623` (Highlight Orange)
- **Background**: White or Light Gray (`#F8FAFC`)

## 2. 레이아웃 및 간격 (Layout & Spacing)
- **로고 여백**: 로고 주변에 최소 `20px` 이상의 안전 여백(Safe Zone)을 유지할 것.
- **컨테이너**: 최대 너비 `1200px` 또는 `1600px` 권장.

## 3. UI 컴포넌트 스타일
- **버튼 및 모서리**: 모든 버튼과 카드 디자인의 `Border-radius`는 `4px`로 고정.
- **그림자**: 부드러운 드롭 섀도우 사용 (`rgba(0, 0, 0, 0.05)`).

## 4. 타이포그래피 (Typography)
- **기본 폰트**: `Noto Sans KR` 우선 적용.
- **폰트 백업**: `sans-serif`.
