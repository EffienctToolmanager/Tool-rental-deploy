# 🧠 안티그래비티 스킬 운영 정책 및 글로벌 내규

## 1. 개요
에이전트의 작업 일관성과 전문성을 유지하기 위해 워크스페이스 내에 자율 스킬(Skills) 시스템을 구축함.

## 2. 스킬 분류 체계
- **Global Scope**: 모든 프로젝트에 공통으로 적용되는 디자인 및 브랜드 내규. (`.agent/skills/global/`)
- **Workspace Scope**: 특정 프로젝트(비품 대여 시스템 등)에 특화된 비즈니스 로직. (`.agent/skills/`)

## 3. 핵심 정책 내용
- **브랜드 정체성**: GEV Blue(`#0047AB`) 및 GE Evergreen 테마를 기본으로 하며, 모든 UI는 화이트 배경의 고밀도 테이블을 지향함.
- **데이터베이스 연동**: 향후 모든 데이터베이스 연동은 **MS 365 SharePoint List**를 표준으로 하며, 명명 규칙(`Checkout`, `Return`, `Tool`)을 엄격히 준수함.
- **지식 영속성**: 모든 작업 결과는 Markdown Table 형식으로 출력하여 옵시디언 지식 베이스와의 호환성을 확보함.

## 4. 기대 효과
- 에이전트 교체 또는 세션 재시작 시에도 동일한 품질과 규칙으로 업무 수행 가능.
- 사용자의 반복적인 지시를 줄이고 자율적인 업무 처리 능력 향상.
