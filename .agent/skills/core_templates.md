# 📄 PDF 리포트 생성 스킬

사용자의 데이터를 분석하여 전문적인 PDF 문서를 생성하는 레시피입니다.

## 사용 도구
- `jspdf`, `html2canvas` (Front-end)
- `reportlab`, `fpdf` (Python Script)

## 실행 절차
1. 데이터 테이블 또는 마크다운 내용을 HTML 템플릿으로 변환.
2. 지정된 폰트(Noto Sans KR) 및 브랜드 컬러 적용.
3. 페이지 레이아웃(A4) 및 머리글/바닥글 설정.
4. 최종 PDF 파일로 워크스페이스에 저장.

# 📂 파일 시스템 제어 스킬

워크스페이스 내의 대규모 파일을 지능적으로 관리하고 구조화합니다.

## 핵심 기능
- **구조 분석**: `tree` 형태의 시각적 맵 생성.
- **배치 처리**: 패턴 매칭을 통한 대량 파일 이동/삭제/이름 변경.
- **무결성 검사**: 중복 파일 감지 및 버전 관리.

# 📊 데이터 시각화 및 차트 스킬

수치 데이터를 기반으로 즉각적인 비즈니스 차트를 생성합니다.

## 지원 차트
- **Line/Area**: 트렌드 분석.
- **Bar/Column**: 비교 분석.
- **Pie/Donut**: 점유율 분석.

## 라이브러리 가이드
- React 환경: `recharts`, `chart.js`.
- 마크다운 환경: `mermaid` 다이어그램 활용.
