# [[Graphify Skill Guide: Codebase Knowledge Graph Generation]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> Graphify is a core analysis skill that generates and visualizes an interactive knowledge graph mapping the complex dependencies and relationships within a codebase.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** True understanding of a codebase's architecture requires transforming physical file structure and function calls into a navigable graph database.
- **세부 내용:**
- **Core Function:** Analyzes a code repository to create a Knowledge Graph, mapping file-to-file and component-to-component relationships.
- **Key Commands:** Uses a CLI for analysis (`update`), continuous monitoring (`watch`), querying graph structure (`query`), and path finding (`path`).
- **Outputs:** Generates three critical artifacts: an interactive visualization (`graph.html`), a summary report (`GRAPH_REPORT.md`), and the raw data structure (`graph.json`).
- **Integration:** Supports linking physical code structure with logical knowledge graphs, useful for tools like Obsidian.
