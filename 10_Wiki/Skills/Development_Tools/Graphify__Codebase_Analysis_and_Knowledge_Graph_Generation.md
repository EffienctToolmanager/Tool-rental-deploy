# [[Graphify: Codebase Analysis and Knowledge Graph Generation]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> Graphify is an essential anti-gravity skill that analyzes a codebase to generate an interactive knowledge graph visualizing complex file relationships.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** Understanding code architecture requires transforming explicit file structures into explicit relationships (a graph representation).
- **세부 내용:**
- Purpose: Analyzes the code base to generate a Knowledge Graph, visualizing complex inter-file relationships.
- Core Functions (CLI): Includes commands for updating the graph, real-time monitoring (`graphify watch`), querying structure (`graphify query`), and finding paths between nodes.
- Outputs: Generates an interactive HTML graph (`graph.html`), a summary report (`GRAPH_REPORT.md`), and the underlying JSON database (`graph.json`).
- Integration: Can be used with Obsidian by generating a tree view, complementing the logical knowledge graph view with the physical file structure.
