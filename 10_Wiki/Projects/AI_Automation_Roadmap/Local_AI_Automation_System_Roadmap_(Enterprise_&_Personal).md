# [[Local AI Automation System Roadmap (Enterprise & Personal)]]

## 📌 한 줄 통찰 (The Karpathy Summary)
> This roadmap outlines the design of an integrated, secure, and localized AI automation system for both professional and personal knowledge management, prioritizing offline capabilities.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** Achieving secure, robust AI automation requires anchoring workflows in local resources (LLMs, Obsidian) and structuring data ingestion via markdown and dedicated, contained modules.
- **세부 내용:**
- **Security Constraint:** The architecture must operate entirely in local/closed-loop environments (e.g., Ollama on local machines) to avoid external cloud dependencies.
- **Enterprise System (Company):** Focuses on file/folder monitoring (Folder Watchdog) for automatic document parsing and data extraction; implementing internal web apps (Streamlit) for common tools.
- **Personal System (Knowledge Management):** Uses Obsidian as the central 'Second Brain' hub; establishing pipelines for daily data ingestion (e.g., investment reports) and ensuring immediate synchronization across devices.
- **Workflow Loop:** Automation requires continuous loops, including generating summarized reports and pushing real-time notifications (Webhook) to maintain awareness and actionability.
