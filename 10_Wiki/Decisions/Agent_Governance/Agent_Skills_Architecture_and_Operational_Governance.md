## 📌 한 줄 통찰 (The Karpathy Summary)
> The policy establishes a comprehensive governance framework for agent skills, enforcing strict rules regarding scope, visual design, data integration, and output formats to ensure consistent professionalism.

## 📖 구조화된 지식 (Synthesized Content)
- **추출된 패턴:** For sustainable automation, agent functionality must be modularized into clearly scoped skills (Global/Workspace) and governed by standardized compliance protocols (Design/Data/Output).
- **세부 내용:**
- Skills must be categorized by scope: Global (universal brand/design rules) or Workspace (project-specific business logic).
- Standard visual identity requires adherence to GEV Blue/Evergreen themes and high-density, white-background table UIs.
- All future database integrations must standardize on MS 365 SharePoint Lists, following strict naming conventions.
- To ensure knowledge persistence, all final work output must be formatted as Markdown Tables for compatibility with the Obsidian knowledge base.
